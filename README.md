# jog-landsend
### CSS animation of a journey from John O' Groats to Land's End.

## Run

**Install [http-server](https://www.npmjs.com/package/http-server)**   
```brew install http-server```

**Change into working directory if not already in it**   
```cd jog-landsend```

**Start http server**   
```http-server -c-1```

**Access**   
```http://localhost:8080```
